## 記事の作成

`_posts`ディレクトリ内に`YYYY-MM-DD-title.md`という命名形式でMarkdownを作成する．
記事の先頭には以下のような`YAML Front Matter`を記述する．

```
---
# YAML Front Matter
title: Markdown语法简介
Category: Misc
tags: 
  - sample, 
  - markdown
toc: true
---

記事本文
```

按照标签和日期查看博文归档。请查看/tags 和/archive 页面。

链接博客主的社交媒体。请在_config.yml中填写。

支持Disqus和Gitalk两种评论区。请在_config.yml中设置。
