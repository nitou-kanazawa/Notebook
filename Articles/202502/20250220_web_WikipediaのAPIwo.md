## Wipediaに用意されているAPI
Wikipediaには以下の3種類のAPIがあるらしい．
- MediaWiki 操作 API
- MediaWiki REST API
- Wikimedia REST API

上の２つはMediawikiに搭載された汎用的なAPI群であり，Wikipedia専用に設計されたものではない．
３つ目の`Wikimedia REST API`はWikipediaを含むWikimedia Projectのために設計されたMediaWikiの拡張機能である．
